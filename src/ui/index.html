Cómo funciona

Este agente corre en tu propio computador o servidor. Lee los contactos de tu base de datos usando tus credenciales locales — estas nunca se envían a Drop Chat. Solo se envían los campos de contacto (nombre, teléfono, email, acciones clave) al API de Drop Chat usando tu API key.
